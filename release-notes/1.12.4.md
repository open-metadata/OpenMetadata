# OpenMetadata 1.12.4 Release Notes

## Improvements

- AI: Add OAuth support for MCP server with database-backed token persistence and automatic refresh [#25391](https://github.com/open-metadata/OpenMetadata/pull/25391)
- AI: Add MCP bot impersonation with audit logging and change event publishing [#26488](https://github.com/open-metadata/OpenMetadata/pull/26488)
- Governance: Implement self-approval prevention for workflow change requests [#26289](https://github.com/open-metadata/OpenMetadata/pull/26289)
- Governance: Add fields filter support in event-based workflows and description change validation node [#26230](https://github.com/open-metadata/OpenMetadata/pull/26230)
- Ingestion: Add Airflow 3.x API-based connector with support for DAG status, configuration, and pipeline management [#26624](https://github.com/open-metadata/OpenMetadata/pull/26624)
- Ingestion: Add support for SAP HANA table function ingestion and lineage [#26711](https://github.com/open-metadata/OpenMetadata/pull/26711)
- Ingestion: Add debug logging for Power BI query parsing source expressions [#26339](https://github.com/open-metadata/OpenMetadata/pull/26339)
- Integration: Add Trino metadata exporter for pushing OpenMetadata descriptions and tags to Trino [#25970](https://github.com/open-metadata/OpenMetadata/pull/25970)
- Platform: Parallelize DataAssetsWorkflow with virtual threads for improved ingestion throughput [#25817](https://github.com/open-metadata/OpenMetadata/pull/25817)

## Fixes

- Discovery: Fix hasDomain() search filter to use OR logic for multi-domain users [#26397](https://github.com/open-metadata/OpenMetadata/pull/26397)
- Data Observability: Fix Trino table diff to use table-specific catalog and schema from FQN instead of service-level defaults [#26604](https://github.com/open-metadata/OpenMetadata/pull/26604)
- Data Observability: Fix tableConstraint to tableConstraints field name in Data Insights config [#26642](https://github.com/open-metadata/OpenMetadata/pull/26642)
- Ingestion: Fix GCS test connection to properly handle excluded buckets [#26627](https://github.com/open-metadata/OpenMetadata/pull/26627)
- Ingestion: Fix Hive ingestion to handle tables with NULL TBL_TYPE in metastore [#26540](https://github.com/open-metadata/OpenMetadata/pull/26540)
- Governance: Fix FQN quoting in entity link comparison for self-approval prevention [#26766](https://github.com/open-metadata/OpenMetadata/pull/26766)
- Lineage: Fix async lineage export silently dropping edges after encountering a missing entity node [#26562](https://github.com/open-metadata/OpenMetadata/pull/26562)
- Platform: Fix race condition in async event consumers on rapid create-delete operations [#26718](https://github.com/open-metadata/OpenMetadata/pull/26718)
- Platform: Add authorization checks to application operational endpoints for trigger and deploy actions [#26590](https://github.com/open-metadata/OpenMetadata/pull/26590)
- Platform: Remove sensitive workflow configuration from TestConnection API responses [#26762](https://github.com/open-metadata/OpenMetadata/pull/26762)
- Platform: Fix UI crash when JSON parsing fails for activity feed data [#26553](https://github.com/open-metadata/OpenMetadata/pull/26553)
- Platform: Fix span tag rendering issue on breadcrumb for metric entities [#26793](https://github.com/open-metadata/OpenMetadata/pull/26793)
- Platform: Fix flaky search index application [#26795](https://github.com/open-metadata/OpenMetadata/pull/26795)
