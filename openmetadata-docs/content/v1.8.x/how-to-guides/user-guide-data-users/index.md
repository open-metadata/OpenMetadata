# Guide for Data Users

This user guide will provide more information on the basics of OpenMetadata, the details of the landing page and the widgets for Activity Feeds, Data Insights, Announcements, and more. Youâ€™ll also get more information about the data assets, and the details associated with it, like tags, tasks, conversations, announcements, versions, and more.

This section will deal with all the details related to data assets: how to change data ownership, how to add tags, glossary terms, announcements, how to follow data assets, how data assets versioning works, and so on.
